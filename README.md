# 🛍️ Customer Behaviour Analysis

## 📌 Project Overview

This project demonstrates a complete industry-standard end-to-end data analytics workflow, covering all key stages from raw data handling to insight generation and reporting.

The objective of this project is to simulate real-world data analyst responsibilities by transforming raw customer data into meaningful business insights and presenting them through interactive dashboards and reports.

The dataset used for this project was sourced from Kaggle and analyzed using Python, PostgreSQL, and Power BI.

---

## 📂 Dataset

**Source:** Kaggle

The dataset contains structured customer-related data used for analysis and visualization. It was imported into Python for preprocessing, exploratory analysis, and dashboard creation.

### 2️⃣ Data Cleaning

Performed data preprocessing using Python:

- Standardized data formats

### 3️⃣ Exploratory Data Analysis (EDA)

Conducted exploratory analysis to identify:

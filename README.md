# Customer_Behaviour_Analysis
I created a Customer Behaviour Analysis report using Python, Sql and PowerBI as a project, using the dataset from kaggle.

📊 End-to-End Data Analytics Project--
 Overview--

This project demonstrates a complete, industry-standard end-to-end data analytics workflow, covering all key stages from raw data handling to insight generation and reporting.The objective is to simulate real-world data analyst responsibilities by transforming raw data into meaningful insights and presenting them through dashboards and reports.

--Dataset--
Source: Kaggle
The dataset contains structured data used for analysis and visualization.
It was imported into Python for preprocessing and further analysis.
🛠️ Tools & Technologies
Python (Pandas) – Data cleaning & EDA
SQL (PostgreSQL) – Data querying & transformation
Power BI – Interactive dashboard creation
Excel – Data handling
Gamma – Presentation (PPT) creation

-- Project Workflow--

1. Data Loading--
Imported dataset from Kaggle into Python environment
Performed initial data inspection

2. Data Cleaning--
Handled missing values
Removed duplicates
Standardized data formats

3. Exploratory Data Analysis (EDA)--
Analyzed data distributions and relationships
Identified trends, patterns, and anomalies

4. SQL Analysis--
Loaded cleaned data into PostgreSQL
Executed SQL queries for deeper insights
Performed aggregations and filtering

5. Dashboard Development--
Built an interactive dashboard in Power BI
Visualized key metrics and KPIs

6. Reporting & Presentation--
Created a detailed analytical report
Designed a presentation using Gamma to communicate insights

Results & Insights--
--Extracted meaningful insights from raw data--
--Identified key trends and business patterns--
--Presented findings in a clear and visual format for stakeholders--
